數據處理條款

最後更新：2026 年 4 月

概述

本數據處理條款說明 Vesta OS 在提供服務過程中如何處理客戶個人數據。若 Vesta OS 根據適用隱私或數據保護法律代表客戶處理個人數據，本條款適用。

雙方角色

對於客戶爲使用服務而提交至 Vesta OS 的客戶個人數據，客戶通常作爲控制者或企業，Vesta OS 通常作爲處理者或服務提供商。

處理範圍

Vesta OS 僅在以下必要範圍內處理客戶個人數據：提供、維護、保護、支持和改進服務；遵守客戶指示；履行合同義務；以及遵守適用法律。

數據類別

客戶個人數據可能包括用戶賬戶信息、住客聯繫方式、訂單信息、住客溝通記錄、物業信息、供應商和任務分配信息、照片、質檢記錄、價格、結算、支持和運營記錄、系統日誌及使用數據。

數據主體類別

數據主體可能包括客戶員工、授權用戶、住客、潛在住客、物業業主、供應商、清潔人員、維修人員、承包商和客服聯繫人。

處理活動

處理活動可能包括託管、存儲、傳輸、檢索、分析、分類、自動化、AI 輔助生成、流程路由、質檢、安全監控、報告和刪除。

客戶指示

Vesta OS 將根據客戶的書面指示處理客戶個人數據，包括協議、產品設置、集成配置和書面說明。

安全措施

Vesta OS 將維護合理的技術和組織措施，以保護客戶個人數據免受未經授權訪問、意外丟失、毀壞、篡改或披露。

子處理者

Vesta OS 可能聘用子處理者，用於提供基礎設施、託管、分析、通信、支持、AI、安全、支付以及運營 Vesta OS 所需的其他服務。

數據返還與刪除

服務終止後，Vesta OS 將根據協議、產品功能和適用法律刪除或返還客戶個人數據，除非因法律、安全、備份或合理業務目的需要繼續保存。

聯繫方式

如對本數據處理條款有任何問題，請通過我們網站提供的聯繫方式與我們聯繫。

本頁面僅供一般信息參考，不構成法律意見。
